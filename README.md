# Ausbildungsbotschafter-Präsentation · Fachlagerist bei DACHSER

Interaktive Web-Präsentation für den IHK-Ausbildungsbotschafter-Einsatz in Schulklassen.

**Muhammad Arifin Abdurrahman** · Fachlagerist, 2. Lehrjahr · DACHSER SE Neumünster

## Ansehen

Die Präsentation liegt als einzelne Datei `index.html` vor. Online über GitHub Pages
oder lokal durch Öffnen der Datei im Browser.

## Bedienung

- **Pfeiltasten ← →** oder die Pfeil-Buttons unten zum Blättern
- **Leertaste** = nächste Folie, **F** = Vollbildmodus
- Interaktive Folien: DACHSER-Quiz, aufklappbare Aufgaben-Karten und die
  Kommissionier-Challenge zum Mitmachen

## Aufbau (nach IHK-Vorgabe)

Einleitung (Aktivierung) → Hauptteil (Beruf, Praxis, persönliche Geschichte) → Schlussteil (Fragen & Kontakt).

## Noch zu ergänzen

- Echte Arbeitsfotos (Lager, Technik, bei der Arbeit) auf der Galerie-Folie
- Zahlen prüfen: Ausbildungsvergütung, Urlaubstage, Berufsschul-Rhythmus
- DACHSER-Eckdaten (Gründungsjahr, Länder) gegenchecken
